自己一直有做网站，之前用的是 hexo 搭建静态博客，无奈发现限制太大，所以想尝试下动态博客，然而网上的教程都要租云服务器，对于我这种常年白嫖的穷鬼来说怎么顶得住，碰巧手头有个树莓派，放着也是浪费，就拿来用用吧～

另外提一句，本教程中除了 vnc 连接的操作都可以在基于 linux 的服务器平台使用，据说宝塔面板喜提 0day 漏洞，所以这个操作应该也能帮到不少人

# 可行性分析：

  • 树莓派的官方系统是基于 linux 的魔改版 debian，所以安装服务器所需配件理论上没有问题
  • 由于我采用的是树莓派 3b+，所以性能方面也无问题
  • 值得一提的是，我们本地的服务器需要做内网穿透才可以进行外部访问，有公网 ip 的小伙伴萌可以不用做这一步
  • 所以继续白嫖，一通寻找，找到了可以白嫖的内网穿透：sakura frp
  • 然后是域名，虽然我依然想白嫖，但无奈 sakura frp 把能白嫖的域名全封禁了，那就只好去腾讯云买个便宜的域名了（悲）
  • ps：记得域名要备案

# 安装树莓派系统

# 所需配件：

  • 树莓派一块（建议选择 3b 以上，不然性能不够）
  • 一个容量大于 10G 的 SD 卡（装系统不需要这么多，但是后期考虑到博客需要存储的东西，可以适当扩充点）
  • 树莓派官方系统镜像
  • 树莓派官方系统烧录软件
  • 读卡器
  • 充电器与充电线（供电用）

# 开始安装

把你的 SD 卡插入读卡器，读卡器插入电脑，打开官方烧录软件

在 CHOOSE OS 中选择你下载的系统镜像（如果是压缩包记得解压，最终是 img 格式），在 CHOOSE SD CARD 中选择你插入的 SD 卡，点击 WRITE

烧录完成后，你的 SD 卡电脑会无法读取，这时不用慌，读卡器拔了再插回去就好了

连接到树莓派有两种方式，我推荐两种方式都用一下，因为操作的时候两种方式各有各的用处
